Instagram Mini Clone (MERN Stack)

A mini Instagram-style web application built using the MERN stack.
This project allows users to create posts, follow other users, like and comment on posts, and view a personalized feed.

Tech Used

React

Node.js

Express.js

MongoDB

JWT Authentication

Features

User signup and login

Password hashing and authentication

Create posts (image URL + caption)

Like and comment on posts

Follow and unfollow users

Feed shows posts from followed users

How to Run

Backend

cd server
npm install
nodemon app.js


Frontend

cd client
npm install
npm start
